# Sams Lawn Care

Marketing website for Sams Lawn Care, providing lawn and landscaping services in Lewisburg, West Virginia and the Greenbrier Valley.

Pages: Home (/), Services (/services), Reviews (/reviews), Work (/work), About (/about), Contact (/contact). Routing is client-side; the build outputs a `_redirects` file and `404.html` fallback so deep links work on Netlify, Cloudflare Pages, and GitHub Pages.

Built with Svelte, Vite, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
